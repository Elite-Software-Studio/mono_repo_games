# Snake Game Development Plan

## 1. Project Setup - Done

- Create project folder and files (`index.html`, `style.css`, `snake.js`)
- Link JS and CSS in HTML

## 2. Canvas Initialization - Done

- Add `<canvas>` element to HTML
- Set canvas size and context in JS

## 3. Game Loop - Done

- Implement main loop using `requestAnimationFrame` or `setInterval`
- Handle game state updates and rendering

## 4. Snake Logic - Done

- Represent snake as an array of coordinates
- Implement movement and direction changes
- Handle snake growth on eating food

## 5. Food Generation - Done

- Randomly place food on canvas
- Detect collision with snake head

## 6. Collision Detection - Pending

- Detect collision with walls and self
- End game on collision

## 7. Input Handling - Done

- Listen for arrow key events to change direction

## 8. Scoring & UI - Pending

- Track and display score
- Show game over message

## 9. Polish & Testing - Pending

- Add restart functionality
- Test for bugs and edge cases

## 10. Optional Features - Pending

- Add sound effects
- Implement difficulty levels
- Add mobile controls
